Matriculación 2024

Iniciamos el Proceso de Inscripciones para la catequesis parroquial 2024-2025. A continuación detallamos lo más importante que debes saber para participar de la catequesis y así despejar las dudas correspondientes.

Requisitos

Nota importante: Para el Proceso de inscripción online es necesario tener la cédula del alumno.

1. Registro web a través del sistema de Gestión de Catequesis que estará habilitado a partir del Martes 20 de Agosto en la web: catequesis.iglesiademachachi.com, de donde se procederá a descargar el Acta de Compromiso que se genera automaticamente una vez concluido el proceso.
2. Acercarse al Despacho Parroquial para la legalización de la Matrícula
Se debe llevar los siguientes documentos:
- Acta de Compromiso impresa y firmada por el representante
- Pase de Nivel

Preguntas Frecuentes

-¿Qué edad debe tener mi hijo/a para ingresar a la Catequesis?

La edad para ingresar a iniciación es de 8 años, bajo una nueva disposición del Arzobispado de Quito, los niños que ingresen a la catequesis deben tener los 8 años cumplidos.

-¿ Si no me salio el acta de compromiso al finalizar el registro en linea? ¿Qué hago?

El Acta de Compromiso se genera automáticamente después de realizar el proceso, por tanto si se realiza en un celular o si se ingresa desde un celular directamente al sitio web por medio de Facebook puede que no se abra o no se descargue correctamente. Probando el sistema no existe errores en este proceso pero la recomendación es usar una computadora o un navegador web para evitar complicaciones.
